# 核酸检测结果管理系统

## 技术选择
- **后端**: Node.js + Express
- **前端**: React.js

### 数据库设计
创建Supabase表：`nucleic_acid_tests`

必需字段：
- `id` (UUID主键)
- `name` (姓名，必填)
- `id_card` (身份证号18位，必填，加索引)
- `test_time` (检测时间，加索引)
- `result` (检测结果：检测中/阴性/阳性，枚举类型)
- `test_location` (检测地点)
- `created_at`（自动时间戳）

要求：调用Supabase MCP，创建表。

## 核心功能

1. **数据库连接模块**
   - 使用环境变量(.env)存储`SUPABASE_URL`和`SUPABASE_KEY`（调用MCP获取）

2. **增加功能 (Create)**
   - 表单输入所有字段
   - 返回成功/失败信息

3. **查询功能 (Read)**
   支持多种查询方式：
   - 按身份证号精确查询
   - 按姓名模糊查询
   - 按手机号查询
   - 按检测结果筛选
   - 按时间范围查询
   - 查询全部（分页，每页10条）
   - 结果按检测时间倒序，表格形式显示

4. **修改功能 (Update)**
   - 所有内容均可修改

5. **删除功能 (Delete)**
   - 支持按ID或身份证号删除
   - 删除前显示记录详情并二次确认
   - 返回操作结果

## 用户界面

- **首页**: 统计仪表板（总人数、阳性率、最近记录）
- **详情页**: 完整信息+编辑/添加/删除按钮
- 使用响应式设计
